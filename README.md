# PacMan2024
The old-fashioned Pac-Man game is now renewed with motivation from Johannes Kepler University
## Getting Started
Simply clone this repository and run the App.java file

## Details
As my first project with the Java Swing library I wanted to implement my own version of the Pac-Man game.
The game depends on a static 2D integer array as its map, which can be updated by other classes.
Everything besides the musics were created/produced by me. (Yes even the cute ghosts and pacman images, thanks paint)
